Document's metadata

Link: <link>

This element allows the incorporation of a local or remote structural or formatting** file to the main HTML document.

Attributes and values:

  • "href": provides identification for** the source of the imported data. It can be:
    • An absolute URL: "https://stylesheets.com/style.css
    • A relative URL: "/styles/style.css"
  • "rel": states the relationship** between the imported document and the main one:
    • The author of the document: "author"
    • The license under which the document is registered: "license"
    • The site's icon: "icon"
    • Previous page: "prev"
    • Next page: "next"
    • Sidebar of the site: "sidebar"
    • Stylesheet used for formatting: "stylesheet"
  • "type": defines the type of the imported document:
    • HTML document: "text/html"
    • CSS document: "text/css"
    • Image document: "image/gif" 
	                
<!DOCTYPE html>

<html>

	<head>

		<link href="/styles/style.css" rel="stylesheet" type="text/css">

		<link href="/documents/document.html" rel="next" type="text/html">

		<link href="/icons/icon.png" rel="icon" type="image/gif">

	</head>

</html>

Script: <script>

This element allows the incorporation of a local or remote behavioral file to the main HTML document.

Attributes and values:

  • "src": provides identification for** the source of the imported data. It can be:
    • An absolute URL: "https://behaviors.com/rules.js
    • A relative URL: "/behavior/rules.js"
  • "type": defines the type of the imported document:
    • JavaScript document: "text/javascript"
  • "async": allows the imported file to be loaded asynchronously. [+]
    • This attribute does not require a value. 
	                
<!DOCTYPE html>

<html>

	<head>

		<script src="/behaviors/rules.css" type="javascript" async>

	</head>

</html>
